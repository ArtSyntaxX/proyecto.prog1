FLUJO DE JUEGO COMPLETO
=======================

1. Ejecutar: python run.py
2. Menu Principal
   - START → Selector de Niveles
   - VESTUARIO → Cambiar nave y balas
   - SALIR → Cerrar aplicacion

3. Selector de Niveles (5 niveles disponibles)
   - Elegir nivel con ARRIBA/ABAJO
   - ESPACIO para seleccionar

4. Juego
   - FLECHAS: Mover nave
   - ESPACIO: Disparar
   - ESC: Volver a menu
   - Objetivo: Alcanzar puntuacion + derrotar todos los enemigos

5. Victoria/Derrota
   - Pantalla final con puntuacion
   - PRESIONA UNA TECLA PARA VOLVER A NIVELES
   - Cualquier tecla devuelve al menu de niveles

6. Repetir desde paso 3 o volver a menu principal

CARACTERISTICAS DEL JUEGO
==========================

5 NIVELES:
- Nivel 1: 3 enemigos, objetiv 30 puntos, movimiento lento
- Nivel 2: 4 enemigos, objetivo 40 puntos, movimiento normal
- Nivel 3: 5 enemigos, objetivo 50 puntos, movimiento rapido
- Nivel 4: 6 enemigos, objetivo 60 puntos, movimiento muy rapido
- Nivel 5: 7 enemigos, objetivo 70 puntos, movimiento extremo

3 TIPOS DE NAVES:
- Nave Verde: Velocidad normal (5)
- Nave Azul: Velocidad rapida (8)
- Nave Roja: Velocidad lenta (3)

3 TIPOS DE BALAS:
- Balas Verdes: Velocidad 8, danio 1
- Balas Rapidas: Velocidad 12, danio 1
- Balas Potentes: Velocidad 5, danio 2

ENEMIGOS:
- Se mueven horizontalmente y disparan hacia la nave
- Cada enemigo tiene vida y se destruye con balas del jugador
- Al destruirlos ganas 10 puntos

PUNTUACION:
- Inicio: 0 puntos
- Por enemigo derrotado: +10 puntos
- Objetivo varia por nivel (30-70 puntos)
- Victoria: Alcanzar objetivo + derrotar todos los enemigos


RESOLUCION TECNICA DE PROBLEMAS
================================

Problema: ImportError en imports relativos
Solucion: Usar run.py como punto de entrada (anade juego/ al path)

Problema: UnicodeEncodeError con caracteres especiales
Solucion: Usar solo caracteres ASCII en prints (sin Unicode)

Problema: Merge conflicts en git
Solucion: git reset --hard HEAD && git clean -fd

Problema: ENTER no funciona en pantalla de derrota
Solucion: Capturar CUALQUIER KEYDOWN, no solo ENTER
